# Netflix Content Analysis and Movie Recommendation System using Machine Learning

## Project Overview

This project analyzes Netflix content data and develops a machine learning-based recommendation system to suggest similar movies and TV shows to users.

The system uses Netflix dataset information such as title, genre, cast, director, release year, rating, and description to understand content patterns and provide recommendations.

## Objective

- Analyze Netflix movies and TV shows data
- Perform data cleaning and preprocessing
- Explore trends using data visualization
- Build a content-based recommendation system
- Recommend similar movies based on user input

## Dataset

Dataset used:
Netflix Titles Dataset

Features include:
- Show type
- Title
- Director
- Cast
- Country
- Release year
- Rating
- Duration
- Genre
- Description

## Technologies Used

- Python
- Google Colab
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Machine Learning Techniques

### TF-IDF Vectorization

TF-IDF converts movie descriptions and genres into numerical values that can be processed by machine learning algorithms.

### Cosine Similarity

Cosine Similarity calculates the similarity between movies and recommends the most related content.

## Project Workflow

1. Load Netflix dataset
2. Data preprocessing
3. Handle missing values
4. Perform Exploratory Data Analysis (EDA)
5. Visualize Netflix content trends
6. Create recommendation system
7. Generate movie recommendations

## How to Run

1. Open Google Colab
2. Upload the dataset file
3. Run all cells in order
4. Enter a movie name
5. Get recommended movies

## Results

The system successfully analyzes Netflix content and provides similar movie/show recommendations based on content features.
